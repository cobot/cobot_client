# CobotClient

A client for the [Cobot](http://cobot.me) [API](http://cobot.me/pages/api) plus helpers.

## Installation

Add this line to your application's Gemfile:

    gem 'cobot_client'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install cobot_client
